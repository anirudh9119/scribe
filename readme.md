# Scribe

Scribe is my implementation of [this](http://arxiv.org/abs/1308.0850) paper by Alex Graves.

It makes use of sequence generators in [blocks](https://github.com/mila-udem/blocks).

Requires:
 * Theano
 * Blocks
 * Fuel

This is work in progress...
